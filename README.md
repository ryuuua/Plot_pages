# Plots Gallery

A self-hosted gallery page for displaying SVG and Plotly HTML plots, designed to match the existing site design.

## Structure

```
plotpages/
├── plots.html              # Main gallery page
├── assets/
│   ├── css/
│   │   ├── main.css        # Base styles matching existing site
│   │   └── gallery.css     # Gallery-specific styles
│   └── js/
│       └── gallery.js      # Gallery rendering and pagination logic
└── plots/                  # Directory for your plot files
    ├── *.svg               # SVG plot files
    └── *.html              # Plotly HTML plot files
```

## Usage

1. **Add your plots** to the `plots/` directory:
   - SVG files: `.svg` format
   - Plotly HTML files: Standalone HTML files generated by Plotly

2. **Update the plots array** in `assets/js/gallery.js`:
   ```javascript
   const PLOTS = [
     { type: "svg",  file: "plots/experiment1.svg",   title: "Experiment 1" },
     { type: "svg",  file: "plots/experiment2.svg",   title: "Experiment 2" },
     { type: "html", file: "plots/trajectory_1.html", title: "Trajectory 1" },
     { type: "html", file: "plots/trajectory_2.html", title: "Trajectory 2" }
     // Add more plots here
   ];
   ```

3. **Configure pagination** (optional):
   - Edit `PLOTS_PER_PAGE` in `assets/js/gallery.js` to change how many plots are shown per page (default: 12)

4. **Deploy to GitHub Pages**:
   - Push the files to your repository
   - Enable GitHub Pages in repository settings
   - The gallery will be available at `https://yourusername.github.io/repository-name/plots.html`

## Features

- **Responsive design**: Works on desktop, tablet, and mobile devices
- **Pagination**: Automatically paginates plots (12 per page by default)
- **SVG support**: Displays SVG files as images
- **Plotly support**: Embeds interactive Plotly HTML plots in iframes
- **Consistent styling**: Matches the existing site design (dark theme, Inter font, purple/green accents)

## Customization

- **Colors**: Edit CSS variables in `assets/css/main.css` (`:root` section)
- **Layout**: Modify grid settings in `assets/css/gallery.css`
- **Pagination**: Adjust `PLOTS_PER_PAGE` in `assets/js/gallery.js`
